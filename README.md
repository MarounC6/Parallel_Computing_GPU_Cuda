# Parallel_Computing_GPU_Cuda realise par CHAHINE Maroun et HABIB Danial

Pour lancer le programme des trois parties, il faut se connecter à une machine de la salle 208 ou 219 du département en ssh en lancant la commande ssh login@if501-219-12,
puis lancer successivement 'make part1', 'make part2' et 'make part3' qui vont compiler les fichiers cuda et lancer les scripts Python qui lancent les éxecutables et créent les graphes dans des fichiers png.

Les scripts Python build_csv créent les fichiers CSV et perf analysis créent les graphes png.
